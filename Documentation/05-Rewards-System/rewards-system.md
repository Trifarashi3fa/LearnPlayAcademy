# LearnPlay Rewards System

The rewards system helps children feel motivated as they learn. Rewards should feel positive, simple, and easy to understand.

## XP Rewards

- Correct answer = 10 XP
- Complete lesson = 50 XP
- Complete level = 100 XP
- Complete mini game = 75 XP
- Complete boss challenge = 250 XP
- Complete world = 500 XP
- 3-star score = bonus reward
- Completion badge for each world

## Mastery Labels

| Progress | Mastery Label |
| --- | --- |
| 0-20% | Beginner |
| 21-40% | Developing |
| 41-60% | Progressing |
| 61-80% | Proficient |
| 81-95% | Advanced |
| 96-100% | Master |

## World Badges

Each world should have one completion badge. A child earns the badge after completing all 10 levels and the boss challenge.

Example:

- Forest World completion badge: Forest Explorer Badge
