# Next Build Steps

These are simple next steps for building the Forest World MVP.

1. Build World Map Screen
2. Load forest-world-config.json
3. Show 10 level nodes
4. Lock levels until previous level is complete
5. Load level question JSON
6. Track correct answers and XP
7. Save progress in localStorage
8. Show parent progress dashboard

## Non-Technical Summary

The app should first show Forest World as a map. A child starts at Level 1. When Level 1 is complete, Level 2 unlocks. The app reads questions from the local JSON files and saves progress on the same device using localStorage.

No database is required for this MVP step.
