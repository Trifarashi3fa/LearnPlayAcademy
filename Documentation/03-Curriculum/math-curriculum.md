# Math Curriculum: Forest World

Forest World is the first LearnPlay Mathematics world. It focuses on primary-level foundations for young learners.

## Forest World Curriculum

| Level | Node Type | Topic |
| --- | --- | --- |
| Level 1 | Learn | Numbers 1-10 |
| Level 2 | Practice | Counting objects |
| Level 3 | Mini Game | Number matching mini game |
| Level 4 | Learn | Simple addition |
| Level 5 | Practice | Addition practice |
| Level 6 | Mini Game | Addition mini game |
| Level 7 | Learn | Simple subtraction |
| Level 8 | Review | Review numbers and operations |
| Level 9 | Challenge | Mixed challenge |
| Level 10 | Boss | Forest Guardian boss quiz |

## Notes For Future Expansion

The same structure can be reused for Mountain World, Ocean World, Space World, and Galaxy World. Each world should include 10 levels, 100 questions, mini games, rewards, and one boss challenge.
