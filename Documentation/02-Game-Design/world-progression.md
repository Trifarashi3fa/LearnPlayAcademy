# LearnPlay Adventure Mode: World Progression

LearnPlay Adventure Mode is the main learning journey for children. Each subject can be turned into themed worlds, levels, mini games, rewards, and boss challenges.

## World Structure

Each world contains:

- 10 levels
- 100 questions
- 5 mini games
- 1 boss challenge
- 1 completion badge

## Learning Worlds

- World 1: Forest World
- World 2: Mountain World
- World 3: Ocean World
- World 4: Space World
- World 5: Galaxy World

| Level | Node Type |

| Level 4 | Learn |
| Level 5 | Practice |
| Level 6 | Mini Game |

| Level 8 | Review |

## How Children Progress

Children begin at Level 1. A level unlocks after the previous level is completed. This keeps the journey simple, guided, and encouraging.

Forest World is designed as the first math adventure. Children learn basic numbers, counting, addition, subtraction, review skills, and then complete the Forest Guardian boss quiz.
