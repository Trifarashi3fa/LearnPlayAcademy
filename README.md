# LearnPlay Academy

Phase 1A front-end prototype for LearnPlay Academy.

LearnPlay Academy is a playful learning website for children aged 7 to 12 and their parents. This prototype includes static pages only. It does not connect to Supabase, Vercel, payments, a database, or authentication.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Pages

- Home
- Subjects
- Games
- About

## Install Dependencies

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Notes

- Game cards are clickable UI prototypes only.
- Backend services, payments, database, authentication, and deployment setup are intentionally excluded from Phase 1A.
