# English Explanation and Hint Standard

Date: 2026-07-19
Project: LearnPlay Academy
Status: Production standard for English Year 1 explanations and hints.

## Purpose

English explanations should feel like a friendly teaching moment, not a dense notes panel. The learner should understand the answer quickly and feel ready for the next question.

## Explanation Standard

Use no more than four primary blocks:

1. Correct answer.
2. Why it is correct.
3. Example or visual.
4. LearnBot tip.

## Optional Blocks

Use optional blocks only when they help the exact question.

- Word meaning: only for genuine vocabulary content.
- Fun fact: only when educationally useful and short.
- Voice button or voice script: where listening or pronunciation supports the skill.

Do not show every optional block on every question.

## Explanation Length Rules

- Correct answer: 1 short line.
- Why it is correct: 1 to 2 short sentences.
- Example or visual: 1 simple example or one clear image/card.
- LearnBot tip: 1 sentence, target 6 to 12 words.
- Avoid paragraphs longer than 2 lines on desktop or mobile.

## Correct Answer Tone

Use brief celebration plus concept reinforcement.

Good patterns:

- "Correct! The word is receive."
- "Great job! The sentence means the cat is resting."
- "Excellent! A matches the beginning sound."

Avoid long praise that hides the teaching point, generic praise repeated every question, and adult terms without explanation.

## Incorrect Answer Tone

Use gentle guidance. Do not shame the learner.

Good patterns:

- "Nice try. Look at the first sound again."
- "Almost there. Read the sentence slowly."
- "Good effort. Check the picture clue."

Avoid:

- "Wrong."
- "You failed."
- "That is easy."
- Any wording that blames the child.

## LearnBot Tip Rules

LearnBot tips should guide strategy, not reveal the answer.

- Maximum: 1 sentence.
- Target length: 6 to 12 words.
- Hard limit: 14 words or about 90 characters.
- Use Year 1 words whenever possible.
- One action per tip.
- Use familiar verbs: look, listen, say, read, match, check.

## Hint Standard

Hints appear before answering. A hint must guide the exact skill being tested without giving away the answer.

| Skill | Good Hint Pattern | Avoid |
| --- | --- | --- |
| Letter recognition | "Look at the letter shape." | Naming the correct letter. |
| Uppercase/lowercase matching | "Find the same letter family." | Showing the exact matching pair. |
| Alphabet sequence | "Say the alphabet slowly." | Saying the missing letter. |
| Phonics | "Listen to the first sound." | Giving the first letter as the answer. |
| CVC blending | "Say each sound, then blend." | Spelling the full word. |
| Vocabulary | "Look at the picture clue." | Giving a meaning that directly identifies the option. |
| Spelling | "Check each letter from left to right." | Saying which option is spelled correctly. |
| Grammar | "Look for the action word or punctuation." | Naming the exact answer label. |
| Sentence order | "Think about who did what." | Giving the full sentence order. |
| Reading comprehension | "Read the sentence again." | Quoting the exact answer line when it reveals the answer. |
| Everyday communication | "Think what someone should say." | Giving the exact phrase to choose. |

## Year 1 Language Level

Use:

- Short sentences.
- Familiar words.
- Concrete actions.
- Present tense where possible.
- Simple terms before grammar labels.

Avoid:

- Abstract explanation before examples.
- Long subordinate clauses.
- Unexplained terms such as predicate, phoneme, adjective, or inference.
- More than one instruction in a hint.

## Prohibited Hint Patterns

A hint must not:

- Include the exact correct answer unless the answer is already visible as the prompt.
- Say "choose A", "choose B", or point to an option position.
- Reveal the first letter when the task asks for the first letter.
- Reveal the missing word when the task is fill in the blank.
- Use the same wording as the correct option when that makes the answer obvious.
- Mention image file names, alt text, or asset labels.
- Depend on colour alone.

## Answer-Revealing Checks

Before approval, check each hint against these questions:

1. Could a child choose the answer without understanding the skill?
2. Does the hint contain the correct answer or a near-copy of it?
3. Does the hint reveal the answer through option order or colour?
4. Does the hint repeat an exact phrase from only one option?
5. Would the hint still be helpful if the options were shuffled?

If any answer is yes, revise the hint.

## Good and Bad Hint Examples

| Skill | Good Hint | Bad Hint |
| --- | --- | --- |
| Letter recognition | Look at the tall and short lines. | Choose lowercase b. |
| Uppercase/lowercase matching | Find the same letter family. | B matches b. |
| Alphabet sequence | Say the alphabet from the start. | The next letter is D. |
| Phonics | Listen to the first sound. | Apple starts with A. |
| CVC blending | Say each sound, then blend. | The word is cat. |
| Vocabulary | Look at what the person is doing. | Sleeping means resting on a mat. |
| Spelling | Check every letter carefully. | Receive has e before i. |
| Grammar | Look at the question mark. | This sentence is a question. |
| Sentence order | Start with who the sentence is about. | Put We first, then played. |
| Reading comprehension | Read the sentence one more time. | The answer is in the second sentence. |
| Everyday communication | Think about polite words. | Say thank you. |

## Voice Script Standard

Voice scripts should:

- Read the prompt naturally.
- Use pauses for blanks.
- Avoid explaining the answer before the learner answers.
- Match the written question exactly unless a spoken version is clearer.
- Stay under 10 seconds for most questions.

## Explanation QA Checklist

Every explanation must pass:

- Correct answer is clear.
- Reason explains the exact skill.
- Example or visual supports the answer.
- LearnBot tip is short and actionable.
- No dense notes panel.
- No repeated generic wording across many questions.
- Optional blocks are used only when relevant.
- Incorrect answer copy is encouraging.
- Text remains readable on mobile.
