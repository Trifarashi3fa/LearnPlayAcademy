# Curriculum

This folder will be the source of truth for LearnPlay subjects, years, topics, subtopics, curriculum sources, learning objectives, and their approval state.

The active MVP remains **Mathematics Year 1 - Forest World**. Phase 3 creates structure only; the website does not read curriculum records from this folder yet.

Future curriculum records should use the shared types in `data/curriculum-types.ts` and should not become active until their linked content has completed the required approval process.
