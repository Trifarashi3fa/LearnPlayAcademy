# Question Bank

This folder will hold governed, playable LearnPlay questions. Future questions must include curriculum mapping, package information, version details, and approval fields before they may appear on the website.

Content may eventually become active only when:

- Status is `Active`.
- Approved By is `Fara`.
- Approval Date is filled.
- The question is not a placeholder.
- The explanation is complete.
- World, Level, and Node Type are filled.
- Package Type is correct.
- MVP Status is correct.

The existing 100 Forest World questions remain in their current location and are not migrated during Phase 3.
