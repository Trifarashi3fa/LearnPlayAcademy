# Local Question Asset Imports

Save exported Google Sheets CSV or XLSX files for Mathematics Year 1 here before running:

```powershell
E:\Codex\npm.cmd run import:questions -- --file "<path-to-export>" --sheet "Forest L01"
```

These exported source files are local review inputs. The importer writes reviewed JSON to the main `content/question-assets/mathematics/year-1/forest-world/` folder.
